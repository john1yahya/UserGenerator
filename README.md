# User Generator

## Description
A simple app that generates random users from a public API. 
You can save your favorites and revisit them anytime.

## Demo
[Live Demo](https://john1yahya.github.io/UserGenerator/card.html)

## Features
- Generate random users with one click
- Save favorite users to a favorites section
- Error handling with retry option
- Data persists with localStorage
